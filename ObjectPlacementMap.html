Geometry Dash Object Placement Map

This is a local spatial debug viewer for Geometry Dash custom level data. Import a full .gmd-style text/XML file, a raw decoded level string, or a text file containing the level data. The viewer extracts placed objects using GD object keys: 1 = object ID, 2 = X, and 3 = Y. Grid lines are every 30 GD units.
