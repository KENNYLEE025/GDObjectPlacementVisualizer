## GD Object Placement Visualizer

Geometry Dash Placement Visualizer is a way for viewers to visualize all objects that are placed in their custom level. To extract GMD levels, use the following website to extract the level contents.

https://gdcolon.com/gdsave/

Features:
- Spatial map viewer
- ID filtering
- GMD to XML (Work in Progress for my custom game engine) 
- Tile and GD Units for objects placed (Will be utilized for my custom game engine)
- Map zooming for clarity (Currently has a max zoom-in percentage relative to the map length and height, so longer levels will not have the best zoom-in outcome)

Please note that this is just the alpha release, and will very likely have some bugs when viewing levels, especially with huge amounts of objects crammed in a small area.
